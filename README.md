# Projeto ETL com Python – DIO Santander

Este projeto foi desenvolvido como parte de um desafio da DIO com foco no entendimento do fluxo ETL (Extração, Transformação e Carregamento).

## Extração
Os dados são lidos a partir de um arquivo CSV contendo informações fictícias de clientes bancários.

## Transformação
Durante a transformação, é gerada uma mensagem personalizada para cada cliente com base em seus dados, utilizando regras simples e/ou um modelo de linguagem (OpenAI).

## Carregamento
O resultado final é salvo em um novo arquivo CSV contendo as mensagens geradas.

## Tecnologias
- Python
- Pandas
- OpenAI API (opcional)
